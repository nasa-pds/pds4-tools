# PDS4 Tools

Tools to generate and use PDS4 metadata.

## Installation

  `npm install pds4-tools -g`
  
## Tools

pds-ldd-doc : Generate Github Flavored Markdown (GFM) documentation for a Local Data Dictionary (LDD) specification file. Suitable for use as a README.md for the LDD.
